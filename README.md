# projeto-inicial-fetch-github-api
OLá Dev, seja bem vindo!
Esse é um projeto que utiliza a API do GitHub para realizar busca de perfis de usrário da plataforma, mostrando os seus ultimos repositórios do GitHub mais um link de acesso a e cada um deles. Fou utilizada a função fetch para realizar busca via https e utilizado funções acingronas para amanipulação de dado.

#Tecnologias usadas
1 - JavaScript
2 - HTML
2 - css
